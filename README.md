# 🍷 ML Wine Classification

This project applies **machine learning techniques** to classify wines based on their chemical properties using a structured dataset.

The work was developed as part of the **Algorithms in AI 3** course and demonstrates the process of **data analysis, preprocessing, model training, and evaluation** using Python.

---

# 🎯 Project Overview

Wine classification is a common supervised machine learning task where the goal is to predict the class of wine based on measurable chemical attributes such as acidity, alcohol content, and other features.

This project explores:

* Data exploration and preprocessing
* Feature analysis
* Training machine learning classification models
* Evaluating model performance

The implementation is carried out in a **Jupyter Notebook environment**, allowing step-by-step experimentation and analysis.
